\begin{abstract}
In dieser Semesterarbeit wird untersucht, ob es gerechtfertigt ist, einen Prototypen
einer Plattform zu erstellen, auf der Medien frei erfasst und gepflegt
werden können. Die Untersuchungen werden anhand des Mediums Film durchgeführt.
Die Plattform soll zudem eine Schnittstelle für andere Programme zur
Verfügung stellen, über die die Medien abgerufen, erstellt und bearbeitetet
werden können.
Auch wird Wert auf die gewählten Technologien und Tools gelegt, damit eine
Weiterentwicklung für Dritte so einfach wie möglich gemacht wird.
\end{abstract}
